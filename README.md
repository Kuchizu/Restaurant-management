# Restaurant Management System

Система управления рестораном на Spring Boot. Монолитное приложение для автоматизации работы ресторана.

## Технологии

- Java 21
- Spring Boot 3.3.5
- PostgreSQL 16
- Gradle
- Liquibase
- JUnit 5, Testcontainers
- OpenAPI / Swagger

## Основные модули

1. **Меню** - управление категориями, блюдами и ингредиентами
2. **Столики** - управление столиками и их статусами
3. **Заказы** - создание заказов, добавление позиций, отправка на кухню
4. **Кухня** - очередь заказов и статусы приготовления
5. **Склад** - учет запасов, резервирование ингредиентов, контроль сроков годности
6. **Поставщики** - управление поставщиками и заказами на поставку
7. **Биллинг** - финализация заказов, чеки, скидки, налоги
8. **Отчеты** - выручка, популярные блюда, рентабельность

## Запуск

### Через Docker

```bash
docker-compose up --build
```

или на более новой версии Docker:

```sh
docker compose up --build
```
